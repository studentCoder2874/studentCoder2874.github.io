## Overview

This is the very site you're reading! Built with Jekyll as a static site generator and Decap CMS for in-browser content management, it's hosted on GitHub Pages with the custom domain `tahirhassan.me`.

## Features

- **Dark Theme** — A sleek dark design with custom CSS properties for easy theming
- **Decap CMS** — Manage all content directly from the browser at `/admin`
- **Fully Responsive** — Mobile-first design that works on all screen sizes
- **Fast** — No CSS frameworks, no unnecessary JavaScript
- **SEO Ready** — Open Graph tags, sitemap, and Jekyll SEO tag plugin

## Tech Stack

- **Jekyll** for static site generation
- **Decap CMS** for content management
- **Pure HTML/CSS/JS** — no frameworks
- **GitHub Pages** for hosting
- **Custom Domain** via CNAME

## What I Learned

Building this site deepened my understanding of the Jekyll templating system (Liquid), CSS custom properties for theming, and how to integrate a headless CMS with a static site generator.
