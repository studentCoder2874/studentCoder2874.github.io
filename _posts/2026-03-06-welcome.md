Welcome to my personal blog! I'm excited to share this space with you.

## What You'll Find Here

This blog is where I'll document my journey as a developer. Expect posts about:

- **Web Development** — Tips, tutorials, and lessons learned from building for the web
- **Project Write-ups** — Deep dives into the projects I work on
- **Technology** — Thoughts on tools, frameworks, and the industry
- **Learning** — Notes from books, courses, and self-study

## About This Site

This site is built with [Jekyll](https://jekyllrb.com), a static site generator, and managed via [Decap CMS](https://decapcms.org). The content lives as Markdown files in a GitHub repository, and GitHub Pages serves the built site at [tahirhassan.me](https://tahirhassan.me).

If you're curious about how it was built, check out the [Portfolio Website project](/projects/sample-project-1/) for details.
